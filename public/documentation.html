Movie API

Welcome to the Movie API documentation. This API provides information about various movies, their directors, and genres. Users can sign up, update their information, and create a list of their favorite movies.

Movie Endpoints

Description Method Endpoint Query Parameters Request Body (Example) Response
Returns a list of all movies GET /movies None None Array of movie objects
Returns movie details by title GET /movies/:title title (movie title) None Movie object with title, description, genre, director, imageURL, featured
Returns genre description by name GET /genres/:genreName name (genre name) None Genre object with name and description
Returns director information by name GET /directors/:directorName name (director's name) None Director object with bio, birth year, death year

User Endpoints

Description Method Endpoint Query Parameters Request Body (Example) Response
Register a new user POST /users None Format: JSON

Example:

{ "username": "john_doe",
"email": "john@example.com",
"password": "secure123" } 		Confirmation message or created user object
Update user information PUT /users/:id username (current username) Format: JSON

Example:

{ "username": "new_name" } 		Updated user object
Add a movie to user’s favorites POST /users/:id/:movieTitle username, movieID None Text: "Movie has been added to user array."
Remove a movie from user’s favorites DELETE /users/:id/:movieTitle username, movieID None Text: "Movie has been removed from users array."
Deregister an existing user DELETE /users/:id username None Text: "User has been removed from database."
