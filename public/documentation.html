Movie API

Welcome to the Movie API documentation. This API provides information about various movies, their directors, and genres. Users can sign up, update their information, and create a list of their favorite movies.

Movie Endpoints






Description Method Endpoint Query Parameters Request Body (Example) Response
Welcome screen GET / None None Welcome message!
Returns a list of all movies GET /movies None None Array of movie objects.
Returns movie details by title GET /movies/:title Movie title None Movie object with title, description, genre, director, actors, imageURL.
Returns genre description by name GET /movies/genre/:genreName Genre name None Genre object with name, description and list of movies under genre.
Returns director information by name GET /movies/directors/:directorName Director's name None Director object with bio, birth year, death year.

User Endpoints








Description Method Endpoint Query Parameters Request Body (Example) Response
Provide list of all users. GET None username None JSON file with list of all users.
Provide individual users. GET /users/:username Username None JSON file with list of single user.
Register a new user POST /users None Format: JSON
{
"username": "alice123",
"password": "password123",
"email": "alice@test.com",
"birthday": "01012001",
"firstName": "Alice",
"lastName": "Bell"
} 		Confirmation message or created user object.
Update user information PUT /users/:username username (current username) Format: JSON

{
username: ,
password: ,
email: ,
birthday: ,
firstName: ,
lastName:
} 		Updated user property.
Add a movie to user’s favorites POST /users/:username/movies/:moviesID username, movieID None Updated JSON file to show added movie.
Remove a movie from user’s favorites DELETE /users/:username/movies/:moviesID username, movieID None Updated JSON file to show deleted movie.
Deregister an existing user DELETE /users/username username None "Username was deleted."
